# Obsidian Navigation Preview

Make Obsidian navigation work like VSCode.

## Features

**VSCode-style tabs**

Single-click a file to open it as a preview (italic, muted title). Single-clicking
another file replaces the preview tab's content. Double-click pins the tab so it
stays open.

## Development

```bash
npm install   # install dependencies
npm run build # type-check + production bundle
```
